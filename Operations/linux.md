# Linux Cheatsheet

## Setup a Firewall with UFW on an Ubuntu and Debian Cloud Server

### TL;DR

- Check status:

```bash
sudo ufw status
```

- Restart (needed when configurations changed):

```bash
sudo ufw status
```

- Allow connections:

```bash
sudo ufw allow ssh
```

```bash
sudo ufw allow 9000/tcp
```
